# NoteX 🚀

A Flutter-based student notes and data management application that allows users to store records and manage information efficiently.

---

## ✨ Features
- 📌 Store student data
- 📝 Manage notes
- ⚡ Fast and simple UI
- 🎯 Easy to use interface

---

## 🛠️ Tech Stack
- Flutter
- Dart

---

## 📱 How to Run
1. Clone the repository
2. Run:
   flutter pub get
3. Start the app:
   flutter run
